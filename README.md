## Clone Repo

If you want to clone this repo, you can add theese commands bellow (i assume you're using Windows for OS).

1. Clone this repo.
```bash
git clone https://github.com/evanalifian/computational-intelligence.git
```

2. Create python virtual environment
```bash
python -m venv myenv
```

3. Install python libraries from **requirements.txt**
```bash
pip install -r requirements.txt
```

## Repo Structure

If you have done clone this repo, the repo structure should be like this:

```bash
├──myenv
├──.gitignore
├──main.py
├──README.md
├──requirements.txt
```
